# Public Assets Folder

Place your static assets here. They will be served from the root path.

## Logo File

Please place your act.3 logo image in this folder as:
- **Filename:** `act3-logo.png` (or `.svg`, `.jpg`, etc.)
- **Path:** `/public/act3-logo.png`
- **Access:** The logo will be accessible at `/act3-logo.png` in your app

The logo should be a black circle with white outlined "act." text and superscript "3".
